### 2.4 唤醒词优先级控制 ✅

```java
// EavesdropperEngine.java
private volatile boolean isWakeupProcessing = false;

public void onWakeupDetected() {
    isWakeupProcessing = true;
    new Handler(Looper.getMainLooper()).postDelayed(() -> {
        isWakeupProcessing = false;
    }, 2000);  // 2秒后恢复
}

public void handleSpeechDetected(...) {
    if (isWakeupProcessing) {
        LogMgr.d(TAG, "Wakeup processing, ignore VAD");
        return;  // ⭐ 唤醒词优先
    }
    
    if (PersonaManager.isShutUpDisabled()) {
        LogMgr.d(TAG, "Shut-up mode active, ignore VAD");
        return;  // ⭐ 闭嘴禁用
    }
}
```

### 2.5 重启默认开启插嘴 ✅

```java
// PersonaManager.java - restorePersonaState()
if (shutUpTime > 0 && System.currentTimeMillis() < shutUpTime) {
    shutUpEndTime = shutUpTime;  // 未过期,恢复闭嘴状态
} else {
    shutUpEndTime = 0;  // ⭐ 已过期或从未设置,默认开启插嘴
    LogMgr.d("PersonaManager", "Eavesdropping enabled by default after reboot");
}
```

## 九、常见问题

### Q1: 编译失败 "JAVA_HOME is set to an invalid directory"

**解决**: 
```bash
# 安装 JDK 8
brew install --cask temurin@8

# 设置 JAVA_HOME
export JAVA_HOME=/Library/Java/JavaVirtualMachines/temurin-8.jdk/Contents/Home
```

### Q2: 闭嘴命令不生效

**检查**:
1. 日志是否有 "Shut-up command detected"
2. `PersonaManager.enableShutUpMode()` 是否被调用
3. `isShutUpDisabled()` 返回值

### Q3: 唤醒词优先级不起作用

**检查**:
1. `EavesdropperEngine.onWakeupDetected()` 是否被调用
2. `isWakeupProcessing` 标志是否正确设置
3. VAD 回调中是否检查了 `isWakeupProcessing`

### Q4: 重启后闭嘴状态丢失

**检查**:
1. `PersonaManager.restorePersonaState()` 是否被调用
2. SharedPreferences 是否正确保存
3. `shut_up_end_time` 键值是否正确
