# 唤醒词与VAD监听冲突解决方案

> **版本**: v48.1  
> **更新日期**: 2026-05-14  
> **问题**: 唤醒词识别与VAD后台监听冲突  
> **解决**: 优先级控制 + 闭嘴命令

---

## 一、问题分析

### 1.1 冲突场景

```
场景: 用户说"成语接龙"(唤醒词)
  ↓
[EavesdropperEngine 正在 VAD 监听]
  ├─ VAD 检测到语音活动
  ├─ 启动 ASR 识别
  └─ 识别结果: "成语接龙"
  ↓
❌ 问题1: EavesdropperHandler 可能先拦截，当成"插嘴"处理
❌ 问题2: 而不是让 NLUDispatcher 当成"唤醒词"处理
❌ 问题3: 用户无法关闭监听功能
```

### 1.2 根本原因

**VAD 监听和唤醒词识别共用同一个麦克风输入**：
- 如果不做优先级控制，VAD 会抢在唤醒词引擎之前处理
- 导致唤醒词被当成普通对话处理
- 用户体验混乱

---

## 二、解决方案

### 2.1 优先级架构设计

```
麦克风音频流
  ↓
[第一层: 唤醒词引擎] ⭐ 最高优先级(始终运行)
  ├─ USC 引擎持续运行
  ├─ 检测到唤醒词 → 立即处理
  ├─ 通知 VAD 引擎暂停
  └─ 未检测到 → 继续下一层
  ↓
[第二层: VAD 监听] (仅在空闲时启用)
  ├─ 检查: 唤醒词是否正在处理?
  │   ├─ 是 → 忽略本次VAD检测
  │   └─ 否 → 继续
  ├─ 检查: 是否在"闭嘴"禁用期?
  │   ├─ 是 → 忽略本次VAD检测
  │   └─ 否 → 继续
  ├─ 检测到语音活动 → 启动 ASR
  └─ 用于"偷听完就插嘴"模式
```

### 2.2 核心机制

#### 机制 1: 唤醒词优先标记

```java
// EavesdropperEngine.java
private volatile boolean isWakeupProcessing = false;

/**
 * 唤醒词检测通知(优先级高于VAD监听)
 */
public void onWakeupDetected() {
    isWakeupProcessing = true;  // ⭐ 标记: 唤醒词正在处理
    
    // 2秒后重置(唤醒词处理完成)
    new Handler(Looper.getMainLooper()).postDelayed(() -> {
        isWakeupProcessing = false;
    }, 2000);
}
```

**VAD 检测时检查**：

```java
vadDetector.start(new VADDetector.VADListener() {
    @Override
    public void onSpeechDetected() {
        // ⭐ 关键检查: 如果唤醒词正在处理，不响应VAD
        if (isWakeupProcessing) {
            LogMgr.d(TAG, "Wakeup processing, ignore VAD");
            return;  // 忽略
        }
        
        // 继续处理...
    }
});
```

#### 机制 2: 闭嘴命令(禁用1小时)

```java
// PersonaManager.java
private static volatile long shutUpEndTime = 0;  // 闭嘴禁用结束时间

/**
 * 启用"闭嘴"模式 - 禁用主动监听1小时

public static void enableShutUpMode() {

 * 检查是否在"闭嘴"禁用期
```

## 三、完整实现

### 3.1 NLUDispatcher 修改

**文件**: `com/unisound/vui/handler/filter/NLUDispatcher.java`

```java
@Override
protected boolean onWakeupResult(ANTHandlerContext ctx, String result) {
    LogMgr.d("NLUDispatcher", "onWakeupResult:" + result);
    LocalASR localASR = this.c.from(result).getLocalASRList().get(0);
    String trim = localASR.getRecognitionResult().trim();
    
    // ⭐ 通知 EavesdropperEngine: 唤醒词正在处理(优先级高于VAD监听)
    if (eavesdropperEngine != null) {
        eavesdropperEngine.onWakeupDetected();
    }
    
    // 判断是否为竞品词
    if (a(ctx.androidContext(), trim)) {
        return true;
    }
    
    // 判断是否为功能唤醒词
    else if (b(ctx.androidContext(), trim) && a(localASR.getScore())) {
        return a(ctx, localASR);
    }
    
    // 判断是否为主唤醒词
    else if (c(ctx.androidContext(), trim) && b(localASR.getScore())) {
        
        // ⭐ 检查是否是"闭嘴"命令
        if (isShutUpCommand(trim)) {
            LogMgr.d("NLUDispatcher", "Shut-up command detected: " + trim);
            
            // 启用闭嘴模式(1小时)
            PersonaManager.enableShutUpMode();
            
            // 播放确认提示
            ctx.playTTS("好的,我闭嘴一小时");
            
            // 记录交互时间
            PersonaManager.recordInteraction();
            
            return true;  // 拦截,不进入对话
        }
        
        // 查找对应的人格配置
        PersonaConfig personaConfig = PersonaConfig.findByWakeupWord(trim);
        if (personaConfig != null) {
            // 发布人格激活事件
            ctx.pipeline().fireUserEventTriggered(
                new PersonaActivationEvent(trim, personaConfig.getPersonaId())
            );
            LogMgr.d("NLUDispatcher", "Published PersonaActivationEvent for: " + trim);
        }
        
        return false;  // 放行,进入语音识别
    }
    
    return true;
}

/**
 * 检查是否是"闭嘴"命令
 */
private boolean isShutUpCommand(String text) {
    return text.contains("闭嘴") || 
           text.contains("别听了") || 
           text.contains("别监听了") ||
           text.contains("安静一小时");
}
```

### 3.2 PersonaManager 修改

**文件**: `com/phicomm/speaker/device/custom/persona/PersonaManager.java`

```java
/** 是否正在监听中(主动模式) */
private static volatile boolean isListening = false;

/** ⭐ 闭嘴禁用结束时间戳 */
private static volatile long shutUpEndTime = 0;

/**
 * 检查是否应该触发主动监听
 */
public static boolean shouldTriggerActiveListening(Context context) {
    // ⭐ 检查是否在"闭嘴"禁用期内
    if (isShutUpDisabled()) {
        long remainingTime = shutUpEndTime - System.currentTimeMillis();
        LogMgr.d("PersonaManager", "Shut-up mode active, remaining: " + (remainingTime/1000) + "s");
        return false;
    }
    
    // 检查当前是否在被动对话中
    if (currentMode == PersonaMode.PASSIVE) {
        long idleTime = System.currentTimeMillis() - lastInteractionTime;
        PersonaConfig activeConfig = PersonaConfig.getActivePersona();
        
        if (idleTime >= activeConfig.getIdleTriggerThresholdMs()) {
            LogMgr.d("PersonaManager", "Idle threshold reached, can trigger active listening");
            return true;
        }
    }
    
    return false;
}

/**
 * 启用"闭嘴"模式 - 禁用主动监听1小时
 */
public static void enableShutUpMode() {
    shutUpEndTime = System.currentTimeMillis() + PersonaConfig.SHUT_UP_DURATION_MS;
    
    // 如果正在监听，立即停止
    if (isListening) {
        stopActiveListening();
    }
    
    // 持久化
    SharedPreferencesHelper.getInstance(context, SP_NAME)
        .saveLongValue("shut_up_end_time", shutUpEndTime);
    
    LogMgr.d("PersonaManager", "Shut-up mode enabled for 1 hour");
}

/**
 * 检查是否在"闭嘴"禁用期
 */
public static boolean isShutUpDisabled() {
    return System.currentTimeMillis() < shutUpEndTime;
}

/**
 * 恢复状态时检查闭嘴状态
 */
public static void restorePersonaState(Context context) {
    // ...原有代码...
    
    long shutUpTime = SharedPreferencesHelper.getInstance(context, SP_NAME)
        .getLongValue("shut_up_end_time", 0);
    
    shutUpEndTime = shutUpTime;  // ⭐ 恢复闭嘴状态
    
    LogMgr.d("PersonaManager", "Restored persona state, shutUpEndTime=" + shutUpEndTime);
}
```

### 3.3 EavesdropperEngine 修改

**文件**: `com/phicomm/speaker/device/custom/engine/EavesdropperEngine.java`

```java
/** ⭐ 唤醒词正在处理中(优先级高于VAD监听) */
private volatile boolean isWakeupProcessing = false;

/**
 * 启动监听引擎
 */
public void start() {
    // ...原有代码...
    
    // 启动 VAD 检测
    vadDetector.start(new VADDetector.VADListener() {
        @Override
        public void onSpeechDetected() {
            // ⭐ 关键检查: 如果唤醒词正在处理，不响应VAD
            if (isWakeupProcessing) {
                LogMgr.d(TAG, "Wakeup processing, ignore VAD");
                return;
            }
            
            // ⭐ 检查是否在"闭嘴"禁用期
            if (PersonaManager.isShutUpDisabled()) {
                long remainingTime = PersonaManager.getShutUpRemainingTime();
                LogMgr.d(TAG, "Shut-up mode active, remaining: " + (remainingTime/1000) + "s");
                return;
            }
            
            handleSpeechDetected();
        }
    });
}

/**
 * ⭐ 唤醒词检测通知(优先级高于VAD监听)
 */
public void onWakeupDetected() {
    isWakeupProcessing = true;
    
    LogMgr.d(TAG, "Wakeup detected, pausing VAD for 2s");
    
    // 2秒后重置(唤醒词处理完成)
    new Handler(Looper.getMainLooper()).postDelayed(() -> {
        isWakeupProcessing = false;
        LogMgr.d(TAG, "Wakeup processing completed, resuming VAD");
    }, 2000);
}
```

### 3.5 ShutUpCommandEvent 事件类

**文件**: `com/phicomm/speaker/device/custom/event/ShutUpCommandEvent.java`

```java
package com.phicomm.speaker.device.custom.event;

/**
 * 闭嘴命令事件
 * 用户说"闭嘴"时触发，禁用主动监听1小时
 */
public class ShutUpCommandEvent {
    private long timestamp;
    private long durationMs;  // 禁用时长
    
    public ShutUpCommandEvent(long durationMs) {
        this.timestamp = System.currentTimeMillis();
        this.durationMs = durationMs;
    }
    
    public long getTimestamp() { return timestamp; }
    public long getDurationMs() { return durationMs; }
    public long getResumeTime() { return timestamp + durationMs; }
    
    @Override
    public String toString() {
        return "ShutUpCommandEvent{duration=" + (durationMs/1000) + "s, resume=" + getResumeTime() + "}";
    }
}
```

## 五、技术要点总结

### 5.1 优先级控制

| 层级 | 组件 | 优先级 | 说明 |
|------|------|--------|------|
| **1** | USC 唤醒词引擎 | ⭐⭐⭐ 最高 | 始终运行,不受任何限制 |
| **2** | VAD 监听引擎 | ⭐⭐ 中 | 受唤醒词和闭嘴命令限制 |
| **3** | ASR 识别 | ⭐ 低 | 仅在 VAD 触发后启动 |

### 5.2 闭嘴命令特性

| 特性 | 说明 | 实现 |
|------|------|------|
| **禁用时长** | 1小时 | `SHUT_UP_DURATION_MS = 3600000` |
| **立即生效** | 停止当前监听 | `stopActiveListening()` |
| **持久化** | 重启后保持 | SharedPreferences 保存 |
| **可恢复** | 1小时后自动恢复 | 时间戳比较 |
| **不影响唤醒词** | 唤醒词始终有效 | 只禁用 VAD |
| **支持多命令词** | 闭嘴/别听了/别监听了 | `isShutUpCommand()` |

### 5.3 安全检查矩阵

| 检查点 | 位置 | 作用 |
|--------|------|------|
| `isWakeupProcessing` | VAD 回调 | 防止 VAD 与唤醒词冲突 |
| `isShutUpDisabled()` | VAD 回调 | 闭嘴期间禁用 VAD |
| `isShutUpDisabled()` | shouldTriggerActiveListening | 防止触发新监听 |
| `onWakeupDetected()` | NLUDispatcher | 标记唤醒词处理中 |

## 七、总结

### 7.1 解决的问题

| 问题 | 解决方案 | 效果 |
|------|---------|------|
| VAD 与唤醒词冲突 | `isWakeupProcessing` 标记 | ✅ 唤醒词优先 |
| 无法关闭监听 | "闭嘴"命令 | ✅ 禁用1小时 |
| 重启后失效 | SharedPreferences 持久化 | ✅ 保持状态 |
| 唤醒词被禁用 | 只禁用VAD,不禁用唤醒词 | ✅ 唤醒词始终有效 |

### 7.2 技术亮点

- ✅ **分层优先级**: 唤醒词引擎 > VAD 监听 > ASR 识别
- ✅ **双重保护**: `isWakeupProcessing` + `isShutUpDisabled()`
- ✅ **用户控制**: "闭嘴"命令,自主决定何时禁用
- ✅ **自动恢复**: 1小时后自动恢复,无需手动操作
- ✅ **持久化**: 重启后保持闭嘴状态

### 7.3 用户体验

```
旧版:
  - VAD 可能干扰唤醒词
  - 无法关闭监听
  - 用户感到被"偷听"

新版:
  - ✅ 唤醒词始终优先
  - ✅ 说"闭嘴"即可关闭1小时
  - ✅ 用户完全掌控
```
