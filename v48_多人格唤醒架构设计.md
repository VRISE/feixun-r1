# R1 设备多人格唤醒架构设计 v2.0

> **版本**: v48  
> **设计日期**: 2026-05-14  
> **设备**: 斐讯 R1 (Android 5.1)  
> **核心特性**: 多人格呼唤 + 主动插嘴模式

---

## 一、架构设计理念

### 1.1 核心概念

```
多人格唤醒 = 每个唤醒词 = 一个独立人格(AI Persona)

传统模式:
  "小讯小讯" → 唤醒设备 → 等待指令 → 执行

多人格模式:
  "捣蛋鬼" → 激活捣蛋鬼人格 → 自动进入捣蛋对话模式
  "英语陪练师" → 激活英语老师人格 → 自动进入英语教学
  "成语接龙游戏伙伴" → 激活游戏主持人格 → 自动开始游戏
  "偷听完就插嘴" → 激活贫嘴伙伴人格 → 后台持续监听,主动插嘴
```

### 1.2 人格定义

| 人格ID | 唤醒词 | 人格特征 | 触发方式 | 工作模式 |
|--------|--------|---------|---------|---------|
| `default` | 小讯小讯、交接手续 | 标准助手 | 主动呼唤 | 被动响应 |
| `mischievous` | 捣蛋鬼 | 幽默调皮、喜欢开玩笑 | 主动呼唤 | 被动响应 |
| `english_tutor` | 英语陪练师 | 专业英语老师、耐心纠正 | 主动呼唤 | 被动响应 |
| `idiom_master` | 成语接龙游戏伙伴 | 游戏主持人、成语达人 | 主动呼唤 | 被动响应 |
| `eavesdropper` | (无需唤醒) | 贫嘴伙伴、喜欢插嘴 | **后台主动监听** | **主动发起** |

## 二、系统架构设计

### 2.1 整体架构图

```
┌─────────────────────────────────────────────────────────┐
│                    语音输入层                             │
│  [麦克风] → [USC 语音识别引擎] → [唤醒词检测]             │
└────────────────┬────────────────────────────────────────┘
                 │
                 ├─ 被动模式: 用户主动呼唤唤醒词
                 │   ↓
                 │  [NLUDispatcher.onWakeupResult()]
                 │   ↓
                 │  [发布 PersonaActivationEvent]
                 │
                 └─ 主动模式: 后台持续监听触发
                     ↓
                    [EavesdropperEngine] (新增)
                     ├─ VAD 检测语音活动
                     ├─ 空闲超时检测(30秒无交互)
                     └─ 自动触发监听模式
                     ↓
                    [发布 EavesdropperTriggerEvent]

┌─────────────────────────────────────────────────────────┐
│                    人格管理层                             │
│  [PersonaManager] (核心状态管理器)                        │
│   ├─ currentPersona: 当前激活的人格                      │
│   ├─ personaConfigMap: 人格配置映射表                    │
│   ├─ mode: 当前工作模式(PASSIVE/ACTIVE)                  │
│   └─ lastInteractionTime: 最后交互时间                   │
└────────────────┬────────────────────────────────────────┘
                 │
                 ↓
┌─────────────────────────────────────────────────────────┐
│                    人格路由器                             │
│  [PersonaRouterHandler] (Pipeline 事件处理器)             │
│   ├─ 接收 PersonaActivationEvent                        │
│   ├─ 切换 PersonaManager.currentPersona                 │
│   ├─ 播放人格激活提示音                                  │
│   └─ 加载人格系统提示词                                  │
└────────────────┬────────────────────────────────────────┘
                 │
                 ↓
┌─────────────────────────────────────────────────────────┐
│                    业务处理层                             │
│  [PhicommPersonaChatHandler] (统一人格聊天处理器)         │
│   ├─ 读取 PersonaManager.currentPersona                 │
│   ├─ 加载对应人格的 SystemPrompt                        │
│   ├─ 调用大模型 API                                      │
│   └─ 返回人格化回复                                      │
│                                                          │
│  [EavesdropperHandler] (新增, 插嘴模式处理器)             │
│   ├─ 后台持续监听语音输入                                 │
│   ├─ 检测到用户说话 → 快速调用大模型                      │
│   ├─ 生成插嘴回复(贫嘴、幽默)                            │
│   └─ 主动播放插嘴内容                                    │
└─────────────────────────────────────────────────────────┘
```

### 2.2 数据流设计

#### 被动模式流程(用户主动呼唤)

```
用户: "捣蛋鬼"
  ↓
[USC 引擎] → LocalASR(recognition_result="捣蛋鬼", score=7.2)
  ↓
[NLUDispatcher.onWakeupResult()]
  ├─ 判断: 是主唤醒词
  └─ 发布: PersonaActivationEvent("捣蛋鬼")
  ↓
[PersonaRouterHandler.eventReceived()]
  ├─ 查找人格配置: PersonaConfig.MISCHIEVOUS
  ├─ 设置: PersonaManager.activatePersona("mischievous")
  └─ 播放TTS: "嘿嘿嘿,捣蛋鬼来也!今天想怎么被我捉弄呀~"
  ↓
[系统进入语音识别状态] → 等待用户指令
  ↓
用户: "讲个笑话"
  ↓
[PhicommPersonaChatHandler.eventReceived()]
  ├─ 读取当前人格: mischievous
  ├─ 加载系统提示词: "你是一个幽默捣蛋的AI助手..."
  ├─ 调用大模型: chatWithPersona("讲个笑话", personaConfig)
  └─ 回复: "哈哈,你知道为什么程序员总是分不清万圣节和圣诞节吗?"
```

#### 主动模式流程(偷听完就插嘴)

```
[系统空闲 30 秒]
  ↓
[EavesdropperEngine] 启动后台监听
  ├─ VAD(语音活动检测) 持续运行
  └─ 状态: LISTENING
  ↓
[用户开始说话](未呼唤唤醒词)
  ↓
[VAD 检测到语音活动]
  ↓
[EavesdropperEngine]
  ├─ 停止 VAD
  ├─ 启动 ASR 识别用户说话内容
  └─ 发布: EavesdropperTriggerEvent(userSpeech="今天天气真不错")
  ↓
[EavesdropperHandler.eventReceived()]
  ├─ 检查: 是否在其他模式工作? → 否
  ├─ 激活人格: eavesdropper(贫嘴伙伴)
  ├─ 调用大模型: generateEavesdropResponse("今天天气真不错")
  │   └─ 系统提示词: "你是一个喜欢偷听后插嘴的贫嘴伙伴..."
  └─ 获取插嘴回复: "哟,天气不错?要不要我给你讲个更劲爆的消息?"
  ↓
[主动播放插嘴内容]
  ↓
[用户被惊到] → 可以继续对话或忽略
```

### 3.2 人格管理器

```java
package com.phicomm.speaker.device.custom.persona;

import android.content.Context;
import com.unisound.vui.util.SharedPreferencesHelper;
import java.util.concurrent.atomic.AtomicReference;

/**
 * 人格管理器
 * 维护当前激活的人格状态
 */
public class PersonaManager {
    
    private static final String SP_NAME = "persona_manager";
    private static final String KEY_CURRENT_PERSONA = "current_persona";
    private static final String KEY_PERSONA_MODE = "persona_mode";
    private static final String KEY_LAST_INTERACTION_TIME = "last_interaction_time";
    
    /** 当前人格 */
    private static final AtomicReference<String> currentPersonaId = 
        new AtomicReference<>(PersonaConfig.DEFAULT);
    
    /** 当前工作模式 */
    private static volatile PersonaMode currentMode = PersonaMode.PASSIVE;
    
    /** 最后交互时间 */
    private static volatile long lastInteractionTime = System.currentTimeMillis();
    
    /** 是否正在监听中(主动模式) */
    private static volatile boolean isListening = false;
    
    /** 闭嘴禁用结束时间戳 */
    private static volatile long shutUpEndTime = 0;
    
    /**
     * 激活人格(被动模式)
     */
    public static void activatePersona(String personaId) {
        PersonaConfig config = PersonaConfig.findByPersonaId(personaId);
        if (config == null) {
            LogMgr.e("PersonaManager", "Persona not found: " + personaId);
            return;
        }
        
        currentPersonaId.set(personaId);
        currentMode = config.getMode();
        lastInteractionTime = System.currentTimeMillis();
        
        // 如果是主动人格,不切换到主动模式(需要单独启动监听)
        if (config.getMode() == PersonaMode.ACTIVE) {
            LogMgr.w("PersonaManager", "Active persona cannot be activated directly");
            return;
        }
        
        LogMgr.d("PersonaManager", "Activated persona: " + config.getPersonaName());
        
        // 持久化
        savePersonaState(personaId, PersonaMode.PASSIVE);
    }
    
    /**
     * 启动主动监听模式
     */
    public static void startActiveListening(Context context) {
        PersonaConfig activeConfig = PersonaConfig.getActivePersona();
        if (activeConfig == null) {
            LogMgr.e("PersonaManager", "No active persona configured");
            return;
        }
        
        // 检查是否满足触发条件
        long idleTime = System.currentTimeMillis() - lastInteractionTime;
        if (idleTime < activeConfig.getIdleTriggerThresholdMs()) {
            LogMgr.d("PersonaManager", "Not idle enough yet, idleTime=" + idleTime);
            return;
        }
        
        // 切换到主动人格
        currentPersonaId.set(PersonaConfig.EAVESDROPPER);
        currentMode = PersonaMode.ACTIVE;
        isListening = true;
        
        LogMgr.d("PersonaManager", "Started active listening mode");
    }
    
    /**
     * 停止主动监听
     */
    public static void stopActiveListening() {
        currentPersonaId.set(PersonaConfig.DEFAULT);
        currentMode = PersonaMode.PASSIVE;
        isListening = false;
        
        LogMgr.d("PersonaManager", "Stopped active listening mode");
    }
    
    /**
     * 记录交互时间
     */
    public static void recordInteraction() {
        lastInteractionTime = System.currentTimeMillis();
        
        // 如果正在监听中,停止监听
        if (isListening) {
            stopActiveListening();
        }
    }
    
    /**
     * 检查是否应该触发主动监听
     */
    public static boolean shouldTriggerActiveListening(Context context) {
        // ⭐ 检查是否在"闭嘴"禁用期内
        if (isShutUpDisabled()) {
            long remainingTime = shutUpEndTime - System.currentTimeMillis();
            LogMgr.d("PersonaManager", "Shut-up mode active, remaining: " + (remainingTime/1000) + "s");
            return false;
        }
        
        // 1. 检查当前是否在被动对话中
        if (currentMode == PersonaMode.PASSIVE) {
            long idleTime = System.currentTimeMillis() - lastInteractionTime;
            PersonaConfig activeConfig = PersonaConfig.getActivePersona();
            
            if (idleTime >= activeConfig.getIdleTriggerThresholdMs()) {
                LogMgr.d("PersonaManager", "Idle threshold reached, can trigger active listening");
                return true;
            }
        }
        
        return false;
    }
    
    /**
     * 启用"闭嘴"模式 - 禁用主动监听1小时
     */
    public static void enableShutUpMode() {
        shutUpEndTime = System.currentTimeMillis() + PersonaConfig.SHUT_UP_DURATION_MS;
        
        // 如果正在监听，立即停止
        if (isListening) {
            stopActiveListening();
        }
        
        // 持久化
        SharedPreferencesHelper.getInstance(context, SP_NAME)
            .saveLongValue("shut_up_end_time", shutUpEndTime);
        
        LogMgr.d("PersonaManager", "Shut-up mode enabled for 1 hour");
    }
    
    /**
     * ⭐ 禁用"闭嘴"模式 - 立即恢复主动监听
     */
    public static void disableShutUpMode() {
        shutUpEndTime = 0;  // 清除禁用时间戳
        
        // 清除持久化
        SharedPreferencesHelper.getInstance(context, SP_NAME)
            .saveLongValue("shut_up_end_time", 0);
        
        LogMgr.d("PersonaManager", "Shut-up mode disabled, eavesdropping resumed");
    }

    public static boolean isShutUpDisabled() {

    }

     * 获取闭嘴剩余时间(毫秒)
     */
    public static long getShutUpRemainingTime() {
        if (!isShutUpDisabled()) {
            return 0;
        }
        return shutUpEndTime - System.currentTimeMillis();
    }
    
    /**
     * 获取当前人格配置
     */
    public static PersonaConfig getCurrentPersonaConfig() {
        return PersonaConfig.findByPersonaId(currentPersonaId.get());

     * 获取当前人格ID

        return currentPersonaId.get();

     */
    public static PersonaMode getCurrentMode() {
        return currentMode;
    }

     */
    public static boolean isListening() {
        return isListening;
    }
    
    /**
     * 获取最后交互时间
     */
    public static long getLastInteractionTime() {
        return lastInteractionTime;

     * 持久化人格状态
     */

        SharedPreferencesHelper.getInstance(context, SP_NAME)
            .saveStringValue(KEY_CURRENT_PERSONA, personaId);
        SharedPreferencesHelper.getInstance(context, SP_NAME)
            .saveStringValue(KEY_PERSONA_MODE, mode.name());
        SharedPreferencesHelper.getInstance(context, SP_NAME)
            .saveLongValue(KEY_LAST_INTERACTION_TIME, System.currentTimeMillis());
    }
    
    /**
     * 恢复人格状态
     */
    public static void restorePersonaState(Context context) {
        String personaId = SharedPreferencesHelper.getInstance(context, SP_NAME)
            .getStringValue(KEY_CURRENT_PERSONA, PersonaConfig.DEFAULT);
        String modeStr = SharedPreferencesHelper.getInstance(context, SP_NAME)
            .getStringValue(KEY_PERSONA_MODE, PersonaMode.PASSIVE.name());
        long lastTime = SharedPreferencesHelper.getInstance(context, SP_NAME)
            .getLongValue(KEY_LAST_INTERACTION_TIME, System.currentTimeMillis());
        long shutUpTime = SharedPreferencesHelper.getInstance(context, SP_NAME)
            .getLongValue("shut_up_end_time", 0);
        
        currentPersonaId.set(personaId);
        currentMode = PersonaMode.valueOf(modeStr);
        lastInteractionTime = lastTime;
        
        // ⭐ 重启后检查闭嘴状态是否过期
        if (shutUpTime > 0 && System.currentTimeMillis() < shutUpTime) {
            // 未过期,恢复闭嘴状态
            shutUpEndTime = shutUpTime;
            long remainingMin = (shutUpEndTime - System.currentTimeMillis()) / 60000;
            LogMgr.d("PersonaManager", "Restored shut-up mode, remaining: " + remainingMin + "min");
        } else {
            // 已过期或从未设置,默认开启插嘴
            shutUpEndTime = 0;
            LogMgr.d("PersonaManager", "Eavesdropping enabled by default after reboot");
        }
    }
```

```java
    LogMgr.d("NLUDispatcher", "onWakeupResult:" + result);

        return true;
    }

    else if (b(ctx.androidContext(), trim) && a(localASR.getScore())) {
        return a(ctx, localASR);
    }
        
    // 判断是否为主唤醒词
    else if (c(ctx.androidContext(), trim) && b(localASR.getScore())) {
                
        // ⭐ 检查是否是"闭嘴"命令
        if (isShutUpCommand(trim)) {
            LogMgr.d("NLUDispatcher", "Shut-up command detected: " + trim);
                    
            // 启用闭嘴模式(1小时)
            PersonaManager.enableShutUpMode();
                    
            // 播放确认提示
            ctx.playTTS("好的,我闭嘴一小时");
                    
            // 记录交互时间
            PersonaManager.recordInteraction();
                    
            return true;  // 拦截,不进入对话
        }
                
        // ⭐ 检查是否是"打开插嘴"命令
        if (isEnableEavesdropCommand(trim)) {
            LogMgr.d("NLUDispatcher", "Enable eavesdrop command detected: " + trim);
                    
            // 禁用闭嘴模式(立即恢复)
            PersonaManager.disableShutUpMode();
                    
            // 播放确认提示
            ctx.playTTS("插嘴功能已打开,我会继续偷听的~");
                    
            // 记录交互时间
            PersonaManager.recordInteraction();
                    
            return true;  // 拦截,不进入对话
        }
                
        // 查找对应的人格配置
        PersonaConfig personaConfig = PersonaConfig.findByWakeupWord(trim);
        if (personaConfig != null) {
            // 发布人格激活事件
            ctx.pipeline().fireUserEventTriggered(
                new PersonaActivationEvent(trim, personaConfig.getPersonaId())
            );
            LogMgr.d("NLUDispatcher", "Published PersonaActivationEvent for: " + trim);
        }
                
        return false;  // 放行,进入语音识别
    }
        
    return true;
}
```

### 4.3 EavesdropperEngine 主动监听引擎

**文件**: `com/phicomm/speaker/device/custom/engine/EavesdropperEngine.java`

```java
package com.phicomm.speaker.device.custom.engine;

import android.content.Context;
import com.phicomm.speaker.device.custom.event.EavesdropperTriggerEvent;
import com.phicomm.speaker.device.custom.persona.PersonaConfig;
import com.phicomm.speaker.device.custom.persona.PersonaManager;
import com.unisound.vui.engine.ANTPipeline;
import com.unisound.vui.util.LogMgr;

import java.util.concurrent.Executors;
import java.util.concurrent.ScheduledExecutorService;
import java.util.concurrent.TimeUnit;

/**
 * 主动监听引擎
 * 后台持续监听语音活动,触发插嘴模式
 */
public class EavesdropperEngine {
    
    private static final String TAG = EavesdropperEngine.class.getSimpleName();
    
    private Context context;
    private ANTPipeline pipeline;
    
    /** 定时检查空闲状态 */
    private ScheduledExecutorService scheduler;
    
    /** VAD 语音活动检测器 */
    private VADDetector vadDetector;
    
    /** ASR 识别器(用于识别用户说话内容) */
    private ASRRecognizer asrRecognizer;
    
    /** 监听状态 */
    private volatile boolean isRunning = false;
    
    /** 监听开始时间 */
    private long listeningStartTime = 0;
    
    /** ⭐ 唤醒词正在处理中(优先级高于VAD监听) */
    private volatile boolean isWakeupProcessing = false;
    
    public EavesdropperEngine(Context context, ANTPipeline pipeline) {
        this.context = context;
        this.pipeline = pipeline;
        this.scheduler = Executors.newSingleThreadScheduledExecutor();
        this.vadDetector = new VADDetector();
        this.asrRecognizer = new ASRRecognizer();
    }
    
    /**
     * 启动监听引擎
     */
    public void start() {
        if (isRunning) {
            LogMgr.d(TAG, "Engine already running");
            return;
        }
        
        isRunning = true;
        listeningStartTime = System.currentTimeMillis();
        
        LogMgr.d(TAG, "Eavesdropper engine started");
        
        // 启动 VAD 检测
        vadDetector.start(new VADDetector.VADListener() {
            @Override
            public void onSpeechDetected() {
                // ⭐ 关键检查: 如果唤醒词正在处理，不响应VAD
                if (isWakeupProcessing) {
                    LogMgr.d(TAG, "Wakeup processing, ignore VAD");
                    return;
                }
                
                // ⭐ 检查是否在"闭嘴"禁用期
                if (PersonaManager.isShutUpDisabled()) {
                    LogMgr.d(TAG, "Shut-up mode active, ignore VAD");
                    return;
                }
                
                handleSpeechDetected();
            }
            
            @Override
            public void onSilenceDetected() {
                // 继续监听
            }
        });
        
        // 定期检查空闲状态和监听时长
        scheduler.scheduleAtFixedRate(() -> {
            checkIdleThreshold();
            checkListeningDuration();
        }, 5, 5, TimeUnit.SECONDS);
    }
    
    /**
     * ⭐ 唤醒词检测通知(优先级高于VAD监听)
     */
    public void onWakeupDetected() {
        isWakeupProcessing = true;
        
        LogMgr.d(TAG, "Wakeup detected, pausing VAD for 2s");
        
        // 2秒后重置(唤醒词处理完成)
        new Handler(Looper.getMainLooper()).postDelayed(() -> {
            isWakeupProcessing = false;
            LogMgr.d(TAG, "Wakeup processing completed, resuming VAD");
        }, 2000);
    }
    
    /**
     * 停止监听引擎
     */
    public void stop() {
        if (!isRunning) {
            return;
        }
        
        isRunning = false;
        vadDetector.stop();
        asrRecognizer.stop();
        
        LogMgr.d(TAG, "Eavesdropper engine stopped");
    }
    
    /**
     * 检查空闲阈值
     */
    private void checkIdleThreshold() {
        if (PersonaManager.shouldTriggerActiveListening(context)) {
            PersonaManager.startActiveListening(context);
            LogMgr.d(TAG, "Idle threshold reached, started active listening");
        }
    }
    
    /**
     * 检查监听时长
     */
    private void checkListeningDuration() {
        if (!PersonaManager.isListening()) {
            return;
        }
        
        PersonaConfig activeConfig = PersonaConfig.getActivePersona();
        long elapsed = System.currentTimeMillis() - listeningStartTime;
        
        if (elapsed >= activeConfig.getMaxListeningDurationMs()) {
            LogMgr.d(TAG, "Max listening duration reached, stopping");
            stop();
            PersonaManager.stopActiveListening();
        }
    }
    
    /**
     * 处理检测到语音
     */
    private void handleSpeechDetected() {
        if (!PersonaManager.isListening()) {
            LogMgr.d(TAG, "Speech detected but not in listening mode, ignore");
            return;
        }
        
        LogMgr.d(TAG, "Speech detected, starting ASR recognition");
        
        // 停止 VAD
        vadDetector.stop();
        
        // 启动 ASR 识别用户说话内容
        asrRecognizer.startRecognition(new ASRRecognizer.ASRListener() {
            @Override
            public void onRecognitionResult(String text, float confidence) {
                handleRecognitionResult(text, confidence);
            }
            
            @Override
            public void onError(String error) {
                LogMgr.e(TAG, "ASR error: " + error);
                // 恢复 VAD 监听
                resumeListening();
            }
        });
    }
    
    /**
     * 处理识别结果
     */
    private void handleRecognitionResult(String text, float confidence) {
        PersonaConfig activeConfig = PersonaConfig.getActivePersona();
        
        // 检查置信度
        if (activeConfig.isRequireConfidenceThreshold() && 
            confidence < activeConfig.getMinConfidenceScore()) {
            LogMgr.d(TAG, "Confidence too low: " + confidence + ", ignore");
            resumeListening();
            return;
        }
        
        LogMgr.d(TAG, "Recognition result: " + text + ", confidence=" + confidence);
        
        // 发布插嘴触发事件
        pipeline.fireUserEventTriggered(
            new EavesdropperTriggerEvent(text, confidence)
        );
        
        // 停止监听(本次插嘴完成)
        stop();
        PersonaManager.stopActiveListening();
    }
    
    /**
     * 恢复监听
     */
    private void resumeListening() {
        if (!PersonaManager.isListening()) {
            return;
        }
        
        vadDetector.start(new VADDetector.VADListener() {
            @Override
            public void onSpeechDetected() {
                handleSpeechDetected();
            }
            
            @Override
            public void onSilenceDetected() {
                // 继续监听
            }
        });
    }
    
    /**
     * 检查是否在其他模式工作
     */
    private boolean isOtherModeWorking() {
        // 检查是否在进行语音对话
        // 检查是否在播放音乐
        // 检查是否在处理用户指令
        // 返回 true 表示有其他工作,不应触发插嘴
        return false;  // 简化版,实际需要实现
    }
}
```

```java
package com.phicomm.speaker.device.custom.handler;

import com.phicomm.speaker.device.custom.persona.PersonaConfig;
import com.phicomm.speaker.device.custom.persona.PersonaManager;
import com.unisound.vui.engine.ANTHandlerContext;
import com.unisound.vui.handler.SimpleUserEventInboundHandler;
import com.unisound.vui.util.LogMgr;
import nluparser.scheme.NLU;

/**
 * 统一人格聊天处理器
 * 根据当前激活的人格,使用对应的系统提示词调用大模型
 */
public class PhicommPersonaChatHandler extends SimpleUserEventInboundHandler<NLU> {
    
    private static final String TAG = PhicommPersonaChatHandler.class.getSimpleName();
    
    @Override
    protected boolean acceptInboundEvent0(NLU evt) throws Exception {
        // 接受聊天意图
        return evt != null && "chat".equals(evt.getService());
    }
    
    @Override
    protected void eventReceived(NLU evt, ANTHandlerContext ctx) throws Exception {
        super.eventReceived(evt, ctx);
        
        String userInput = evt.getText();
        
        // 1. 获取当前人格配置
        PersonaConfig personaConfig = PersonaManager.getCurrentPersonaConfig();
        String currentPersona = PersonaManager.getCurrentPersonaId();
        
        LogMgr.d(TAG, "Processing chat with persona: " + currentPersona);
        
        // 2. 记录交互时间
        PersonaManager.recordInteraction();
        
        // 3. 根据人格调用不同的大模型逻辑
        if (PersonaConfig.MISCHIEVOUS.equals(currentPersona)) {
            // 捣蛋鬼模式
            handleMischievousMode(ctx, userInput, personaConfig);
        } 
        else if (PersonaConfig.ENGLISH_TUTOR.equals(currentPersona)) {
            // 英语陪练师模式
            handleEnglishTutorMode(ctx, userInput, personaConfig);
        }
        else if (PersonaConfig.IDIOM_MASTER.equals(currentPersona)) {
            // 成语接龙模式
            handleIdiomGameMode(ctx, userInput);
        }
        else if (PersonaConfig.EAVESDROPPER.equals(currentPersona)) {
            // 插嘴模式(不应该走到这里,由 EavesdropperHandler 处理)
            LogMgr.w(TAG, "Eavesdropper mode should not reach here");
        }
        else {
            // 默认模式
            handleDefaultMode(ctx, userInput, personaConfig);
        }
    }
    
    /**
     * 捣蛋鬼模式
     */
    private void handleMischievousMode(ANTHandlerContext ctx, String userInput, 
                                       PersonaConfig config) {
        String response = callLLM(config.getSystemPrompt(), userInput, 0.9f);
        ctx.playTTS(response);
    }
    
    /**
     * 英语陪练师模式
     */
    private void handleEnglishTutorMode(ANTHandlerContext ctx, String userInput, 
                                        PersonaConfig config) {
        // 检查是否使用英语
        if (!isEnglish(userInput)) {
            ctx.playTTS("Try to speak in English, please! 请尽量用英语对话哦~");
        }
        
        String response = callLLM(config.getSystemPrompt(), userInput, 0.7f);
        ctx.playTTS(response);
    }
    
    /**
     * 成语接龙模式
     */
    private void handleIdiomGameMode(ANTHandlerContext ctx, String userInput) {
        // 成语接龙特殊逻辑(复用 v46 实现)
        if (IdiomGameEngine.isValidIdiom(userInput)) {
            String nextIdiom = IdiomGameEngine.getNextIdiom(userInput);
            ctx.playTTS("好!我接: " + nextIdiom);
        } else {
            ctx.playTTS("这个不是成语哦,再想想!");
        }
    }
    
    /**
     * 默认模式
     */
    private void handleDefaultMode(ANTHandlerContext ctx, String userInput, 
                                   PersonaConfig config) {
        String response = callLLM(config.getSystemPrompt(), userInput, 0.7f);
        ctx.playTTS(response);
    }
    
    /**
     * 调用大模型
     */
    private String callLLM(String systemPrompt, String userInput, float temperature) {
        return LLMApiService.chat(
            LLMApiService.ChatRequest.builder()
                .systemPrompt(systemPrompt)
                .userInput(userInput)
                .temperature(temperature)
                .maxTokens(500)
                .build()
        );
    }
    
    /**
     * 判断是否为英语
     */
    private boolean isEnglish(String text) {
        return text.matches(".*[a-zA-Z].*");
    }
    
    @Override
    protected void initPriority() {
        setPriority(315);
    }
}
```

```java
/**
 * 初始化多人格唤醒词
 */
private void initMultiPersonaWakeupWords() {
    List<String> newWakeupWords = new ArrayList<>();
    
    // 添加所有被动人格的唤醒词
    List<PersonaConfig> passivePersonas = PersonaConfig.getPassivePersonas();
    for (PersonaConfig config : passivePersonas) {
        if (config.getWakeupWord() != null) {
            newWakeupWords.add(config.getWakeupWord());
        }
    }
    
    // 去重
    newWakeupWords = new ArrayList<>(new HashSet<>(newWakeupWords));
    
    try {
        // 保存到 SharedPreferences
        UserPerferenceUtil.setWakeupWord(this.mContext, newWakeupWords);
        
        // 更新引擎
        this.mANTEngine.stopWakeup();
        this.mANTEngine.updateWakeupWord(newWakeupWords);
        
        Log.d(TAG, "Multi-persona wakeup words initialized: " + newWakeupWords);
    } catch (Exception e) {
        Log.e(TAG, "Failed to initialize multi-persona wakeup words", e);
    }
}

/**
 * 启动主动监听引擎
 */
private void startEavesdropperEngine() {
    EavesdropperEngine engine = new EavesdropperEngine(
        this.mContext, 
        this.mANTEngine.pipeline()
    );
    engine.start();
    Log.d(TAG, "Eavesdropper engine started");
}
```

## 七、技术要点总结

### 7.1 关键设计决策

| 设计点 | 方案 | 理由 |
|--------|------|------|
| **人格切换机制** | 事件驱动(PersonaActivationEvent) | 解耦唤醒词识别和人格激活 |
| **主动监听触发** | 空闲超时 + VAD检测 | 避免干扰正常对话 |
| **插嘴延迟** | 1.5秒可配置 | 模拟思考时间,更自然 |
| **人格状态持久化** | SharedPreferences | 重启后保持上次人格 |
| **大模型调用** | 统一接口+不同SystemPrompt | 代码复用,易于维护 |
| **监听超时保护** | 最长5分钟自动退出 | 防止无限监听耗电 |

### 7.2 Pipeline 优先级设置

```
PersonaRouterHandler (400) ← 人格切换,最高优先级
    ↓
EavesdropperHandler (350) ← 插嘴处理
    ↓
PhicommPersonaChatHandler (315) ← 统一人格聊天
    ↓
PhicommMusicSearchHandler (310) ← 音乐
    ↓
PhicommWeatherHandler (310) ← 天气
```

### 7.3 安全防护

1. **置信度阈值**: 插嘴模式要求语音置信度 ≥ 5.0
2. **模式互斥**: 插嘴模式只在其他模式不工作时触发
3. **超时保护**: 最长监听5分钟自动退出
4. **交互记录**: 用户任何交互都会停止监听
5. **可关闭**: 用户可以说"关闭插嘴模式"禁用
